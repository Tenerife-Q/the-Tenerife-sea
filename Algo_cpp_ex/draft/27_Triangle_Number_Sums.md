# Triangle Number Sums

## 题目描述

**Time limit:** 1.00 s  
**Memory limit:** 512 MB

三角形数（Triangle Number）是形如 $1 + 2 + \cdots + k$ 的正整数。前几个三角数为 1, 3, 6, 10, 15。

**关键事实：** 每个正整数都可以表示为多个三角数的和。例如：
- $42 = 21 + 21$
- $1337 = 1326 + 10 + 1$

**问题：** 给定正整数 $n$，求最少需要多少个三角数相加得到 $n$。

### 输入

第一行：整数 $t$（测试用例数）  
接下来 $t$ 行，每行一个正整数 $n$

### 约束

- $1 \le t \le 100$
- $1 \le n \le 10^{12}$

### 样例

**输入:**
```
5
1
2
3
42
1337
```

**输出:**
```
1
2
1
2
3
```

### 样例说明

- $n=1$: 1 本身是三角数，答案为 1
- $n=2$: $2 = 1 + 1$，答案为 2
- $n=3$: 3 本身是三角数，答案为 1
- $n=42$: $42 = 21 + 21$，答案为 2
- $n=1337$: 需要 3 个三角数，答案为 3

---

## 数学原理与证明

### 1. 三角数的定义与性质

**定义：** 第 $k$ 个三角数为：
$$T_k = 1 + 2 + 3 + \cdots + k = \frac{k(k+1)}{2}$$

**性质：**
- 首几个三角数：$T_1=1, T_2=3, T_3=6, T_4=10, T_5=15, T_6=21, \ldots$
- 三角数快速增长，但比平方数稍慢

### 2. 检验三角数的方法

给定 $n$，判断其是否为三角数。需要解方程：
$$\frac{k(k+1)}{2} = n$$

即：
$$k^2 + k - 2n = 0$$

由二次公式：
$$k = \frac{-1 + \sqrt{1 + 8n}}{2}$$

**结论：** $n$ 是三角数当且仅当 $8n + 1$ 是完全平方数，且相应的 $k$ 是正整数。

**验证举例：**
- 对于 $n=6$：$8 \times 6 + 1 = 49 = 7^2$，$k = \frac{-1+7}{2} = 3$，所以 6 是三角数 ✓
- 对于 $n=2$：$8 \times 2 + 1 = 17$，不是完全平方数，所以 2 不是三角数 ✓

### 3. Legendre 定理与推广

**Legendre 四平方定理：** 每个正整数都可以表示为至多 4 个整数的平方和。

**推广到三角数：** 存在类似的定理说明每个正整数都可以表示为至多 3 个三角数的和。

**证明思路（意义描述）：**

对于任意正整数 $n$，我们分三种情况：

1. **情形 1：$n$ 本身是三角数**
   - 直接得到答案为 1

2. **情形 2：$n$ 可以表示为 2 个三角数的和**
   - 枚举所有三角数 $T_k \le n$
   - 检查 $n - T_k$ 是否也是三角数
   - 若存在这样的 $T_k$，答案为 2

3. **情形 3：$n$ 需要 3 个三角数**
   - 根据推广的 Legendre 定理，如果 $n$ 既不是三角数，也不能表示为 2 个三角数的和
   - 那么 $n$ 必定能表示为 3 个三角数的和

**关键观察：** 由于约束中 $n$ 最大可达 $10^{12}$，我们需要高效地检验。三角数增长快（约 $\sqrt{2n}$ 个三角数小于等于 $n$），所以枚举是可行的。

### 4. 数学证明梗概

虽然完整证明较复杂，但基本思路是：

**Theorem (三角数表示定理)：** 对于任何正整数 $n$：
- 若 $n \equiv 0 \pmod{4}$ 或 $n \equiv 3 \pmod{4}$，则 $n$ 可以表示为 1 或 2 个三角数的和
- 若 $n \equiv 1, 2 \pmod{4}$ 且不是三角数，则 $n$ 可以表示为 2 或 3 个三角数的和
- 总之，**任何正整数最多需要 3 个三角数**

这与 Legendre 四平方定理的结构类似，但应用到三角数领域。

## 解法代码分析

```cpp
bool isPerfectSquare(long long n) {
    if (n < 0) return false;
    long long sr = sqrt(n);
    return sr * sr == n || (sr + 1) * (sr + 1) == n;
}
```

**用途：** 检验 $n$ 是否为完全平方数，需要考虑浮点精度问题（检查 $\lfloor\sqrt{n}\rfloor$ 和 $\lceil\sqrt{n}\rceil$）

```cpp
bool isTriangleNumber(long long n) {
    return isPerfectSquare(8 * n + 1);
}
```

**用途：** 利用 $8n + 1$ 为完全平方数的充要条件来判断 $n$ 是否为三角数

```cpp
int countMinTriangleNumbers(long long n) {
    // 步骤 1：检查是否本身是三角数
    if (isTriangleNumber(n)) {
        return 1;
    }
    
    // 步骤 2：检查能否表示为 2 个三角数的和
    for (long long k = 1; k * (k + 1) / 2 <= n; k++) {
        long long triangleK = k * (k + 1) / 2;
        if (isTriangleNumber(n - triangleK)) {
            return 2;
        }
    }
    
    // 步骤 3：根据定理，如果前两步都不成立，则答案为 3
    return 3;
}
```

**时间复杂度：**
- 每个三角数 $T_k = \frac{k(k+1)}{2} \approx \frac{k^2}{2}$
- 对于 $n \le 10^{12}$，最多约 $\sqrt{2 \times 10^{12}} \approx 1.4 \times 10^6$ 个三角数
- 枚举时间：$O(\sqrt{n})$
- 整体：$O(t \times \sqrt{n})$，满足时间限制

## 核心思想总结

1. **利用充要条件判断三角数：** $n$ 是三角数 $\iff$ $8n + 1$ 是完全平方数
2. **逐层检验：** 先检查 1 个，再检查 2 个，最后根据定理返回 3
3. **数学保证：** Legendre 推广定理保证答案不会超过 3
4. **效率优化：** 三角数虽多，但仍在可枚举范围内（$O(\sqrt{n})$ 个）

### 第一步：到底什么是“三角形数”？（看图就懂）

古希腊数学家毕达哥拉斯学派喜欢用“石子”在沙滩上摆形状。如果能把石子刚好摆成一个**等边三角形**，那么这些石子的总数就叫“三角形数”。

我们可以画图来看看：

**第 1 个三角形数 ($T_1$)：1 个石子**
```text
*
（总数：1）
```

**第 2 个三角形数 ($T_2$)：底边 2 个，上面 1 个**
```text
  *
 * *
（总数：1 + 2 = 3）
```

**第 3 个三角形数 ($T_3$)：底边 3 个，依次往上**
```text
   *
  * *
 * * *
（总数：1 + 2 + 3 = 6）
```

**第 4 个三角形数 ($T_4$)：**
```text
    *
   * *
  * * *
 * * * *
（总数：1 + 2 + 3 + 4 = 10）
```

**发现规律了吗？**
第 $k$ 个三角形数，其实就是从 $1$ 一直加到 $k$。
高斯小时候算过的 $1+2+3+...+100$，算出来的其实就是第 100 个三角形数！

根据等差数列求和公式，**第 $k$ 个三角形数的公式是：**
$$ T_k = \frac{k(k+1)}{2} $$

题目问：**随便给你一个整数 $n$，你最少需要挑几个“三角形数”，把它们加起来等于 $n$？**
（允许挑一样的，比如 $6 = 3 + 3$，用了 2 个）

---

### 第二步：为什么答案最多只能是 3？（配方法的奇迹）

要想知道 $n$ 能不能由 3 个三角形数拼成，我们假设它能：
$$ n = T_a + T_b + T_c $$
把公式代进去：
$$ n = \frac{a(a+1)}{2} + \frac{b(b+1)}{2} + \frac{c(c+1)}{2} $$
（注意，这里的 $a,b,c$ 可以是 0，因为 $T_0 = 0$，表示少用一个）

这个式子带分数很难受，我们怎么把它变成熟悉的数学式子呢？
**初中数学教过我们“配完全平方”：$(2x+1)^2 = 4x^2 + 4x + 1$**

为了凑出这个形式，我们在等式两边同时**乘以 8**：
$$ 8n = 4a(a+1) + 4b(b+1) + 4c(c+1) $$
展开括号：
$$ 8n = 4a^2 + 4a + 4b^2 + 4b + 4c^2 + 4c $$
为了配成完全平方，每一项都需要加个 1。这里有 3 项，所以等式两边同时**加 3**：
$$ 8n + 3 = (4a^2 + 4a + 1) + (4b^2 + 4b + 1) + (4c^2 + 4c + 1) $$
把完全平方式缩写回去：
$$ 8n + 3 = (2a+1)^2 + (2b+1)^2 + (2c+1)^2 $$

**惊天大发现：**
“$n$ 能不能变成 3 个三角形数之和” 这个问题，等价于：
**“$8n + 3$ 能不能写成 3 个奇数的平方和？”**

这时候，**勒让德三平方和定理（Legendre's three-square theorem）** 登场了。
这个定理证明了：**除了一种极其特殊格式的数字以外，其他所有正整数，统统可以写成三个平方数之和。**
这种被拉黑的特殊格式是：$4^k(8m + 7)$ （也就是一直除以 4，最后除以 8 余 7 的数）。

我们来看看 $8n + 3$：它除以 8 的余数永远是 **3**，根本不可能余 7！
所以，$8n+3$ 绝对不属于那种“拉黑名单”，它**一定**可以写成 3 个平方数的和。

又因为 $8n+3$ 是奇数，只有“奇数+奇数+奇数”才能得到奇数，所以这三个平方数的底数一定都是奇数，完美符合 $(2a+1)^2$ 的形式！

**结论：因为 $8n+3$ 永远能拆成三个奇数的平方，所以倒推回去，$n$ 永远能拆成三个三角形数之和。答案永远不超过 3！**

---

### 第三步：如何判断答案是不是 1？

既然最多是 3，我们只需要依次检查 1 和 2 即可。

如果 $n$ 本身就是 1 个三角形数：
$$ n = \frac{k(k+1)}{2} $$
像刚才一样，两边乘 8，但这次为了配平方，只要**加 1** 就可以了：
$$ 8n = 4k^2 + 4k $$
$$ 8n + 1 = 4k^2 + 4k + 1 = (2k+1)^2 $$

**判断方法：**
只要算一下 $8n + 1$ 开根号是不是个整数（即是否是完全平方数）。如果是，说明 $n$ 就是三角形数，答案是 1。

---

### 第四步：如何判断答案是不是 2？（重点推导费马定理）

如果 $n$ 是 2 个三角形数之和：
$$ n = \frac{a(a+1)}{2} + \frac{b(b+1)}{2} $$
两边乘 8，这次有两项，所以**加 2**：
$$ 8n + 2 = (2a+1)^2 + (2b+1)^2 $$

**核心问题来了：一个整数 $M$ (这里是 $8n+2$)，什么时候能写成两个数字的平方和 ($X^2 + Y^2$)？**

我们来推导一下。假设有一个数 $p$ 可以写成 $X^2 + Y^2$。
我们知道，任何一个偶数，它的平方除以 4 余 0（比如 $2^2=4, 4^2=16$）。
任何一个奇数，它的平方除以 4 永远余 1（比如 $3^2=9=4\times2+1, 5^2=25=4\times6+1$）。

所以，一个完全平方数除以 4 的余数，只能是 **0 或者 1**。
那么，两个平方数相加（$X^2 + Y^2$）除以 4 的余数，只能是以下三种情况之一：
- $0 + 0 = 0$
- $0 + 1 = 1$
- $1 + 1 = 2$
**它绝对不可能等于 3！**

这意味着：**所有除以 4 余 3 的质数（比如 3, 7, 11, 19），它们自己绝对不可能写成两个平方数之和！**

那假如一个大数字里包含了除以 4 余 3 的质因子呢？
伟大数学家费马给出了**“费马平方和定理”**：
一个大数字在分解质因数后，那些“除以 4 余 3”的“坏质数”，如果你想让大数字变成两平方和，**这些坏质数必须成双成对地出现！**

举个例子：
- 质数 3 （坏质数，3 除以 4 余 3）：不能写成两平方和。
- 但是 $3^2 = 9$：它可以！ $9 = 3^2 + 0^2$ （因为平方本身就是一个平方数加 0）。
- 如果是 $3^3 = 27$：不行！

**判断方法：**
1. 我们把 $8n+2$ 拿来分解质因数。
2. 遇到一个质因数 $d$ 时，检查它是不是坏质数（即 $d \pmod 4 == 3$）。
3. 如果是坏质数，看它出现了几次。如果是奇数次，直接宣判死刑——它不可能写成两平方和，意味着 $n$ 不可能写成 2 个三角形数之和！

---

### 总结整个逻辑

对于任何输入的数字 $n$：
1. **测试答案 1：** 算 $8n+1$，如果是完全平方数，答案就是 `1`，结束。
2. **测试答案 2：** 算 $8n+2$，给它分解质因数。如果所有满足除以 4 余 3 的质数都出现了偶数次，说明它可以变出两平方和，答案就是 `2`，结束。
3. **兜底答案 3：** 如果前面都不对，根据高斯的定理，它最多就是 `3`，不用算了，直接输出 `3`。
