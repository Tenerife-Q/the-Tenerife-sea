# 组合数学专题：错排问题 (Derangement) 全解析

错排问题是组合数学中一个既有递推美感，又有容斥逻辑的经典模型。本文集合了从基础定义到深层数学原理的完整推导过程。

---

## 1. 核心定义与直觉误区

### 什么是错排？
**定义：** 对于 $n$ 个元素，没有任何一个元素处于它原本的位置上。
* **例子：** $n$ 个人分 $n$ 个礼物，每个人都**绝对不能**拿到属于自己的那个礼物。

### 澄清误区：关于 "n-1"
* **误区：** “错排的定义是不是只能选择 $n-1$ 个可能？”
* **真相：** 只能选择 $n-1$ 个可能是解决问题时的**第一步限制**（因为第 $n$ 个人不能选自己的）。但“错排”的本质是**全局性**的——要求全员都不在位，而不仅仅是第一个人的选择受限。

---

## 2. 递推公式一：分类讨论视角

**公式：** $D_n = (n - 1) \times (D_{n-1} + D_{n-2})$

### 推导过程
假设有 $n$ 个孩子，编号为 $1, 2, \dots, n$，礼物编号也是 $1, 2, \dots, n$。

1.  **第一步：** 考虑**第 $n$ 个孩子**。他不能拿 $n$ 号礼物，所以他有 **$(n-1)$** 种选择。假设他拿了第 $k$ 号礼物。
2.  **第二步：** 考虑**第 $k$ 个孩子**。此时分两种情况：
    * **情况 A：第 $k$ 个孩子刚好拿了第 $n$ 号礼物**
        * 这相当于 $n$ 与 $k$ 完成了互换。剩下的 $n-2$ 个孩子分剩下的 $n-2$ 个礼物，且必须错排。
        * 方案数：$D_{n-2}$。
    * **情况 B：第 $k$ 个孩子没有拿第 $n$ 号礼物**
        * 第 $n$ 个人已拿走 $k$ 的礼物离开。剩下 $n-1$ 个人和 $n-1$ 件礼物。
        * 由于规定“第 $k$ 个人不能拿 $n$ 号”，我们可以把 $n$ 号礼物看作是 $k$ 的“新原配”，这等价于 $n-1$ 个元素的错排。
        * 方案数：$D_{n-1}$。

**结论：** 总方案数 $D_n = (n - 1)(D_{n-1} + D_{n-2})$。

---

## 3. 递推公式二：代数变形视角

**公式：** $D_n = n \times D_{n-1} + (-1)^n$

### 推导过程
通过对公式一进行代数变形，可以得到只需前一个状态的递推式：

1.  **拆解公式一：**
    $$D_n = n D_{n-1} - D_{n-1} + (n - 1) D_{n-2}$$
2.  **移项构造对称式：**
    $$D_n - n D_{n-1} = - (D_{n-1} - (n - 1) D_{n-2})$$
3.  **构造数列：**
    令 $A_n = D_n - n D_{n-1}$，则有 $A_n = - A_{n-1}$。
4.  **求通项：**
    这是一个公比为 $-1$ 的等比数列，$A_n = A_2 \times (-1)^{n-2}$。
    已知 $D_1=0, D_2=1$，则 $A_2 = D_2 - 2D_1 = 1$。
    故 $A_n = (-1)^n$。
5.  **还原：**
    $$D_n = n D_{n-1} + (-1)^n$$

---

## 4. 底层原理：容斥原理 (Inclusion-Exclusion)

容斥原理是错排公式非递归形式的来源，其核心逻辑是：**“多退少补”**。

### 数学逻辑步进
为了求“全错排”，我们从全集中剔除所有不合格的方案：

1. **全集：** $n!$（没有任何限制，随手乱分）。
2. **第一次修正（减法）：** 减去“至少有一个人在原位”的情况。选 1 个人在位有 $C(n, 1)$ 种，剩余人乱排有 $(n-1)!$ 种。
   $$\text{修正量} = C(n, 1) \times (n-1)! = \frac{n!}{1!}$$
3. **第二次修正（加法）：** 上一步中，“同时有两个人原位”的情况被减了两次，补回一次。
   $$\text{修正量} = C(n, 2) \times (n-2)! = \frac{n!}{2!}$$
4. **以此类推（交替加减）：** $$D_n = n! - \frac{n!}{1!} + \frac{n!}{2!} - \frac{n!}{3!} + \dots + (-1)^n \frac{n!}{n!}$$

**通项公式：**
$$D_n = n! \left( \sum_{k=0}^{n} \frac{(-1)^k}{k!} \right) = n! \left( 1 - \frac{1}{1!} + \frac{1}{2!} - \frac{1}{3!} + \dots + \frac{(-1)^n}{n!} \right)$$

### 为什么会出现 $(-1)^n$？
这个符号在公式中充当了“加减开关”。它反映了在处理重叠集合时，为了保证每个错排方案被计数且仅被计数一次，必须交替进行排除和补回的操作。

---

## 5. 错排概率与自然对数 $e$

当 $n \to \infty$ 时，括号内的级数正是 $e^x$ 在 $x=-1$ 时的泰勒展开：
$$\lim_{n \to \infty} \frac{D_n}{n!} = \frac{1}{e} \approx 0.3678$$
这告诉我们：在一个大型随机分配中，所有人都不拿到自己原配礼物的概率稳定在 **36.8%** 左右。

# 组合数学与算法实现：错排问题 (Derangement) 全解析

错排问题是组合数学中极具代表性的模型，也是算法竞赛中常见的考点。本文整合了错排的基础定义、两种数学推导逻辑，以及标准 C++ 竞赛级别的代码实现。

---

## 1. 核心定义

**定义：** 对于 $n$ 个元素，没有任何一个元素处于它原本的位置上。
* **通俗表达：** $n$ 个人分 $n$ 个属于他们自己的东西，每个人都**绝对不能**拿到属于自己的那一个。

---

## 2. 宏观上帝视角：容斥原理 (Inclusion-Exclusion)

官方题解的 **Solution 1** 使用了反向思考的容斥原理：不直接求错排，而是求出**非错排**（至少有人拿对自己礼物）的方案数，再用总数 $n!$ 减去它。

### 数学逻辑
定义 $f(n, i)$ 为：在 $n$ 个元素中，恰好选定 $i$ 个元素在自己本来的位置上，剩下 $n-i$ 个元素**随意排列**的方案数。
计算公式为：
$$f(n, i) = \binom{n}{i} (n - i)!$$
* $\binom{n}{i}$：表示从 $n$ 个人中选出 $i$ 个人拿对自己的礼物。
* $(n - i)!$：表示剩下的 $n - i$ 个人随便分剩下的礼物。

根据容斥原理的“多退少补”，我们对 $i$ 从 $1$ 到 $n$ 进行遍历：奇数时相加（剔除），偶数时相减（补回被多剔除的重叠部分）。

### C++ 竞赛实现解析 (Solution 1)

在大数计算中，题目通常要求对 $10^9+7$ 取模。这在代码实现中引入了经典的数论技巧：

1. **组合数取模除法转化为乘法（乘法逆元）：**
   因为取模运算 $(A / B) \pmod M$ 是不成立的，我们需要根据**费马小定理**，将除以 $B$ 转化为乘以 $B$ 的逆元 $B^{M-2} \pmod M$。
2. **快速幂 (`power`)**：用于在 $O(\log N)$ 时间内高效计算 $B^{M-2}$。

```cpp
#include <iostream>
using namespace std;
using ll = long long;

const int M = 1000000007;
const int N = 1000000;

// 快速幂算法：求 a^b % M
int power(int a, int b) {
    if (b == 0) return 1;
    ll u = power(a, b / 2);
    u = u * u % M;
    if (b % 2 == 1) u = u * a % M;
    return u;
}

// 费马小定理求乘法逆元
int inv(int x) {
    return power(x, M - 2);
}

ll fac[N + 1];

// 计算组合数 C(n, k) % M
int ncr(int a, int b) {
    // a! / (b! * (a-b)!) 转化为 a! * inv(b!) * inv((a-b)!)
    return fac[a] * inv(fac[b] * fac[a - b] % M) % M;
}

int main() {
    // 1. 预处理阶乘数组，O(N)
    fac[0] = 1;
    for (int i = 1; i <= N; i++) {
        fac[i] = fac[i - 1] * i % M;
    }

    int n;
    cin >> n;
    ll count = 0;
    
    // 2. 容斥原理：计算非错排的总数
    for (int i = 1; i <= n; i++) {
        ll add = ncr(n, i) * fac[n - i] % M;
        if (i % 2 == 0) add = -add; // 奇数加，偶数减
        count = (count + add) % M; 
    }
    
    // 3. 用总排列数 n! 减去非错排总数
    // 注意 C++ 中负数取模的坑，需要 + M 再 % M 来保证结果为正
    ll answer = ((fac[n] - count) % M + M) % M;
    cout << answer << "\n";
}
```

## 3. 微观身临其境：动态规划递推

官方题解的 **Solution 2** 给出了递推公式的代码实现，这与我们之前的分类讨论完美契合：

**核心公式：** $$d(n) = (n - 1) \times (d(n - 2) + d(n - 1))$$

### 题解逻辑视角的精妙之处
对于“情况 2”（即第 $y$ 个孩子把礼物给了除 $x$ 之外的其他人），官方题解给出了一个**插入法**的精彩解释：
如果不考虑孩子 $x$，剩下的 $n-1$ 个孩子进行错排，有 $d(n-1)$ 种方案。现在我们把孩子 $x$ 强行**插入**到这个稳定的错排系统中：让 $x$ 把礼物给 $y$，然后原本给 $y$ 送礼物的那个人，改把礼物送给 $x$。
因为有 $n-1$ 个不同的 $y$ 可以作为切入点，所以这部分构成了 $(n-1) \times d(n-1)$。

### C++ DP 实现 (Solution 2)

这个解法的时间复杂度为 $O(N)$，且避开了复杂的乘法逆元和组合数计算，代码极其优雅。

```cpp
#include <iostream>
#include <vector>
using namespace std;
using ll = long long;

const int M = 1000000007;

int main() {
    int n;
    cin >> n;

    // DP 状态数组
    vector<ll> count(n + 1);
    
    // Base cases
    if (n >= 1) count[1] = 0;
    if (n >= 2) count[2] = 1;
    
    // 状态转移
    for (int i = 3; i <= n; i++) {
        count[i] = (i - 1) * (count[i - 2] + count[i - 1]) % M;
    }
    
    cout << count[n] << "\n";
}
```
