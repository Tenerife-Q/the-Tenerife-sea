# 链地址法哈希表测试分析报告

## 1. 测试环境与配置

- **哈希函数**: 除留余数法 (mod 13)
- **默认表大小**: 13 (可配置，建议10-20)
- **允许装载因子**: ≤ 3.0（每个链表平均长度不超过3）

## 2. 测试场景说明

### 2.1 随机冲突测试
```c
// 测试数据特点
- 数据范围：[0, 999]
- 数据量：50个键值
- 分布特征：完全随机
```

**预期结果**：
- 冲突分布相对均匀
- ASL接近理论值
- 链长差异不大

### 2.2 伪随机测试
```c
// 线性同余生成器参数
seed = 12345
next = (seed * 1103515245 + 12345) & 0x7fffffff
value = next % 1000
```

**特点**：
- 可重现的随机序列
- 固定的分布模式
- 适合性能对比

### 2.3 聚集测试
```c
// 测试数据
起始值：100
数据量：50
序列：[100, 101, 102, ..., 149]
```

**目的**：
- 验证最坏情况下的性能
- 测试连续键值的分布
- 评估冲突处理效率

## 3. 性能指标分析

### 3.1 装载因子(α)影响
| 装载因子 | 期望ASL | 典型实测ASL |
|---------|---------|------------|
| 0.25    | 1.125   | 1.1 - 1.2  |

### 3.2 测试结果分析要点

1. **随机测试**
   - 关注ASL与理论值的吻合度
   - 观察链长分布的均匀性
   - 验证随机分布的冲突处理效果

2. **伪随机测试**
   - 对比不同种子的结果
   - 评估序列的周期性影响
   - 检验结果的可重复性

3. **聚集测试**
   - 分析连续键值的分布
   - 评估最大链长
   - 观察冲突处理的效率

## 4. 日志解读指南

### 4.1 日志格式
```
========== 测试记录 [时间戳] ==========
测试类型: [类型名称]
关键字数量: N
装载因子: α
实际ASL: actual_asl
理论ASL: 1 + α/2
最大链长: max_len
总冲突次数: collisions
=====================================
```

### 4.2 关键指标说明

1. **装载因子(α)**
   - α < 0.5: 空间利用率低
   - 0.5 ≤ α ≤ 0.75: 理想区间
   - α > 0.75: 注意性能下降

2. **ASL对比**
   - 实际ASL ≈ 理论ASL: 表现正常
   - 实际ASL > 理论ASL: 需要优化
   - 差异 > 20%: 可能存在问题

3. **最大链长**
   - ≤ 3: 良好
   - 4-5: 可接受
   - > 5: 需要关注
