=== 交互式二次探测法哈希表使用演示 ===

1. 启动程序：.\interactive_quadratic_probe_en.exe

2. 主菜单选项：
   =====================================
   Quadratic Probing Hash Table Menu
   =====================================
   1. Insert element        - 插入元素
   2. Search element        - 查找元素  
   3. Delete element        - 删除元素
   4. Display hash table    - 显示哈希表
   5. Reset hash table      - 重置哈希表
   6. Batch insert test data - 批量插入测试数据
   7. Exit program          - 退出程序

3. 使用示例流程：

   步骤1：选择6 - 批量插入测试数据
   这将插入预设的测试数据：21, 32, 10, 43, 54, 65, 76, 87, 98, 99, 11, 22, 33, 44
   
   步骤2：选择4 - 显示哈希表
   查看当前哈希表状态和冲突解决结果
   
   步骤3：选择1 - 插入元素
   输入一个新的数字，观察插入过程和冲突处理
   
   步骤4：选择2 - 查找元素
   输入要查找的数字，观察查找过程
   
   步骤5：选择3 - 删除元素
   输入要删除的数字，观察删除过程
   
   步骤6：选择5 - 重置哈希表
   清空所有数据，重新开始
   
   步骤7：选择7 - 退出程序

4. 程序特点：
   - 详细显示每次操作的探测过程
   - 实时显示冲突解决步骤
   - 支持插入、查找、删除、重置操作
   - 表格化显示哈希表状态
   - 自动计算装载因子
   - 使用与原文件相同的测试数据

5. 冲突解决演示：
   程序会详细显示二次探测的每一步：
   - 初始哈希位置
   - 每次探测的位置
   - 冲突检测和解决过程
   - 最终插入位置

6. 查找过程演示：
   - 显示查找的起始位置
   - 逐步显示探测序列
   - 明确指出找到或未找到的结果
